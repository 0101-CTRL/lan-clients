This patch reduces ambiguous timestamp/activity wording:

- Activity Inference -> Freshness estimate
- Recent record update -> Record updated recently
- Timestamp fallback badge -> simple “NCM record update / No router report time” text
- Router report rows show “Router report time / Best available client-report timestamp”
- Tooltip wording is simplified and more explicit about stored NCM data vs live connected state
